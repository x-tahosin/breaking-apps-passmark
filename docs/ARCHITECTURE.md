# Architecture

- `01-02`: HOCKS AI (auth + chat)
- `03`: Vercel Commerce (e-commerce)
- `04`: Cal.com (scheduling)
- `05`: Hoppscotch (API testing)
- `06`: Documenso (document signing)
- `07`: EcoSense AI (green tech)
